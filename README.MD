#### Run The App Locally

```sh
npm run install-dependencies
```

- rename .env.temp to .env
- setup values for - MONGO_URL, JWT_SECRET, JWT_LIFETIME, CLOUD_NAME, CLOUD_API_KEY, CLOUD_API_SECRET

```sh
npm start
```

- visit url http://localhost:3000/
